# MM Library 🚀

Repositório do sistema criado a partir do projeto da disciplina de Programação Orientada a Objetos do Curso de ADS - UESB

## 📚 O que estou praticando aqui:
* **Lógica de Programação**
* **Java Fundamentals:** Estruturas de repetição, condicionais e tipos de dados.
* **POO (Programação Orientada a Objetos):** Classes, objetos, herança e polimorfismo.
* **Futuramente** Conexão de sistemas Java com MySQL para salvar em um BD ao invés de um .txt

  ## 🛠️ Tecnologias Utilizadas:
* Java JDK 17+
* IDE NetBeans
* Git & GitHub

* ## 📝 Como executar os arquivos:
1. Clone o repositório.
2. Abra o projeto no NetBeans.
3. Localize o arquivo `MMLibrary.java` dentro do pacote de fontes.
4. Clique com o botão direito e selecione **Run File**.
